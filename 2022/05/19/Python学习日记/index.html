发表于 2022-05-19 分类于 学习日记

pyhhon的“规矩”非常少，上手极为简单，正因如此，在学习过其他语言比如C++后再来学习这个，会有一定程度上的麻烦。

基础的函数 打印 print("内容") 输如 input("提示")

变量名可以用中文，不可以用数字和下划线开头，区分大小写

分支结构

快速运算符

简写 含义
x += 2 x = x+2
x *= 2 x = x*2
x % y 求x/y的余数
x // y 求 x/y的商
x ** y x的y次幂

字符串需要用 ""或者 ''标起来。如果字符串内用到 ""，那字符串就用 ''标记，如果字符串内两个都用到了， 用 "''"三引号标记
